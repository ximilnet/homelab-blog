What is Proxmox?

Proxmox is a hypervisor—software that lets you run multiple virtual machines and containers on a single physical server. Think of it as a hotel building. The hotel is the physical server. The rooms are your containers and virtual machines.

LXC vs VM: What's the Difference?

FeatureVirtual Machine (VM)LXC Container
KernelOwn kernel (more isolation)Shares host kernel
ResourcesHeavy (more RAM, disk)Lightweight (less RAM, disk)
Startup timeMinutesSeconds
OS supportAny (Windows, BSD, Linux)Linux only
Use caseStrong isolation, different OSFast, efficient Linux services
💡 Which should you choose?
For Windows or BSD → use a VM.
For Linux services (WireGuard, Jellyfin, Nextcloud) → use an LXC container.

Creating Your First LXC

Step 1: Download a Template

In Proxmox go to local storage → CT Templates → Download. Choose Debian 12.

Step 2: Create the Container

  • General: ID (e.g., 106), hostname (e.g., jellyfin-demo), root password
  • Template: Select Debian 12
  • Root Disk: 8 GB (4 GB for basic services)
  • CPU: 2 cores (1 is enough for lightweight services)
  • Memory: 2 GB (512 MB for basic containers)
  • Network: Static IP, gateway (e.g., 192.168.1.1), DNS (e.g., 192.168.1.1)
  • Features: ✅ Unprivileged container (security) and ✅ Nesting (for Docker/systemd)

Inside the Container – First Commands

Enter the container console (right-click → Console) and run:

# Check what OS is inside
cat /etc/os-release

# Update package list
apt update

# Upgrade installed packages
apt upgrade -y

Install Jellyfin (Demo)

# Add Jellyfin repository
curl -fsSL https://repo.jellyfin.org/debian/jellyfin_team.gpg.key | gpg --dearmor -o /etc/apt/keyrings/jellyfin.gpg
echo "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/jellyfin.gpg] https://repo.jellyfin.org/debian bookworm main" | tee /etc/apt/sources.list.d/jellyfin.list

# Update and install
apt update
apt install jellyfin -y

Open your browser and go to http://<LXC_IP>:8096. The Jellyfin setup page should appear.

Command Definitions for Beginners

  • cat – Concatenate; reads and displays file contents
  • apt – Advanced Package Tool; package manager for Debian/Ubuntu
  • curl – Client URL; transfers data over the internet
  • gpg – GNU Privacy Guard; verifies digital signatures
  • | (pipe) – Sends output from one command to another
  • -y – Automatically answers "yes" to confirmation prompts
🔒 OPSEC Note: Always check Unprivileged container – it isolates the container from the Proxmox host. Keep containers updated with apt update && apt upgrade regularly.
